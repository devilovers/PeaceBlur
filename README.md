## Overview

PeaceBlur is an experimental computer vision project that combines real-time hand tracking and image processing to create an interactive camera experience. The application detects hand gestures and dynamically applies a blur effect, demonstrating the integration of machine learning-based tracking with Python desktop applications.

---

## Features

* Real-time webcam processing
* Hand gesture detection
* Dynamic blur effect control
* MediaPipe Hands integration
* Interactive computer vision experience
* Lightweight Python implementation
